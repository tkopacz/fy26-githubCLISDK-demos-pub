# SampleProject (dane wejściowe dla demo 16)

Fikcyjny projekt wejściowy SDLC dla demo z wieloma agentami.

Intencjonalnie zawiera:
- niepełne wymagania,
- znaczniki TODO,
- uproszczony przepływ autoryzacji,
- podstawowy podział na serwis i repozytorium.

Etap architekta powinien delegować analizę specjalistyczną i połączyć ją z deterministycznymi sprawdzeniami statycznymi.
